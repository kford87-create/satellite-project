Four meaningfully distinct visual directions for Quotebird. Each is a self-contained HTML file with three sections: marketing landing, signup / paywall, and the main app screen (Prompt-to-Quote on mobile). Click any card to open.

Direction 01

Warm Trades-Modern

Filson-catalog-meets-Notion. Parchment, forest green, brick orange. Refined serif headlines with italic accents. Feels like a shop that respects the craft and updated the tools.
Direction 02

Bold Hi-Contrast

Stripe/Linear toughened. Stark white, near-black ink, one strong copper accent. Oversize condensed display type. Confident, declarative, modern tooling.
Direction 03

Editorial Trade Journal

Monocle-for-trades. Paper cream, deep navy, rust accent. Magazine layout with masthead, decks, pull quotes, and a rate card. Quotebird as a publication people want to read.
Direction 04

Confident Dark Mode

Premium pocket tool. Charcoal background, warm cream text, single amber accent. Humanist sans throughout. Every pixel intentional — feels like you're holding a precision instrument.
